Laravel Rewardable

Installation

First, pull in the package through Composer.

$ composer require draperstudio/laravel-rewardable:^1.0

And then include the service provider within app/config/app.php.

'providers' => [
    DraperStudio\Rewardable\ServiceProvider::class
];

At last you need to publish and run the migration.

php artisan vendor:publish --provider="DraperStudio\Rewardable\ServiceProvider" && php artisan migrate

Setup a Model

<?php

namespace App;

// use DraperStudio\Rewardable\Traits\Badgeable as BadgeableTrait;
// use DraperStudio\Rewardable\Traits\Creditable as CreditableTrait;
// use DraperStudio\Rewardable\Traits\Rankable as RankableTrait;
// use DraperStudio\Rewardable\Traits\Transactionable as TransactionableTrait;
use DraperStudio\Rewardable\Traits\Rewardable as RewardableTrait;
use Illuminate\Database\Eloquent\Model;

class User extends Model
{
    use RewardableTrait;

    // these can be required one-by-one if you don't need all and don't use RewardableTrait
    // use BadgeableTrait;
    // use CreditableTrait;
    // use RankableTrait;
    // use TransactionableTrait;
}

Classes and Methods

Credits

Get all Credits of a user

$user->credits();

Get all Credits of a user

$user->getCredit();

Get all Credits of a user by Credit type

$user->getCreditByType($type);

Get the current balance of a user

$user->getBalance();

Get the current balance of a user by Credit type

$user->getBalanceByType($type);

Get all Credits a user has spent

$user->getSpendCredits();

Grant a single Credit

$user->grantCredit(Credit::find($id));

Grant multiple Credits

$user->grantCredits([
    Credit::find($id), Credit::find($id), Credit::find($id)
]);

Revoke a single Credit

$user->revokeCredit(Credit::find($id));

Revoke multiple Credits

$user->revokeCredits([
    Credit::find($id), Credit::find($id), Credit::find($id)
]);

Revoke all credits

$user->revokeAllCredits();

Regrant all Credits (Removes all credits and assigns the new ones) 

$user->reGrantCredits([
    Credit::find($id), Credit::find($id), Credit::find($id)
]);

Create a new Credit Type

CreditType::create(['name' => 'Participation']);

Transactions
